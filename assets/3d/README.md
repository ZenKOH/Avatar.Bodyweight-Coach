# 3D Asset Folder

Place prepared GLB/VRM/FBX/BVH candidate assets here only after licence review.

Do not commit third-party avatars, textures, rigs or animations unless redistribution rights are clear. Add attribution and licence notes for every asset in `ATTRIBUTION.md` before bundling.

Preferred app-ready format: `.glb` with named animation clips.

Suggested named animation clips:

